## 环境准备

首先得有 [node](https://nodejs.org/en/)，并确保 node 版本是 10.13 或以上。

```bash
$ node -v
v10.13.0
```

## 安装

建立一个空文件夹，然后在文件夹下执行以下命令进行安装：

```bash
$ npm i dumi -D
```

## 开始写文档

dumi 默认会自动搜寻 `docs`、`src`（或各 lerna 包下的 `src`）目录下的 markdown 文件
